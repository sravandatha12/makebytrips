# SkyKings Flight Booking App

A premium, modern flight booking web application built with React and Vite.

## Features
- **Smart Search**: Real-time flight search with dates and passenger count.
- **Flight Results**: Filter by non-stop interactive list with airline details.
- **Seat Selection**: Visual seat map with business/economy class indication.
- **Passenger Details**: seamless form flow.
- **Digital Boarding Pass**: Aesthetic ticket generation.

## Tech Stack
- React 18
- Vite
- Lucide React (Icons)
- Vanilla CSS (Glassmorphism design system)

## Getting Started

1. Install dependencies (if not already done):
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open your browser at `http://localhost:5173`.
